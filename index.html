LjSlider - javascript based Slider v1.1

To fork or download? here is the GitHub Repo

Getting started

To start implement an html template like this one below
<div class="myclass"></div>
Then, no more thing to do than calling the function : Basic use
Ps: The jQuery plugin is coming soon

Basic use

                    new LjSlider({
                        elt: document.querySelector('.myclass'),
                    });

Full documentation

Attribute and class

  • data-val *attribute
    Set the slider default values
    <div class="myclass" data-val="-40.6">
    <div class="myclass" data-val="-40.6,7,9,3,0">
    * To make it work, don't precise a values parameter to your LjSlider parameters.
    Go to --> Value parameter <--
  • onlight *class
    Add this class to your element if you're not using a dark background
    <div class="myclass onlight">

Parameters

  • elt * Required
    The DOM element which is going to be the slider line
    Type: DOMElement
    Default value: null
    Value examples: Go to --> Basic use <--
  • handler * Optional
    Is a function called when the slider values change, shoould take an ordered array as single parameter
    Type: function
    Default value: console.log
    Value examples: Go to --> Changing handler <--
  • link * Optional
    Show the green background link if true
    Type: Boolean
    Default value: true
  • pins * Optional
    THe slider pins (handles) number
    Type: Number (int)
    Default value: 1
    Value examples: 6, 3
  • sameStep * Optional
    Set a same distance between steps if true
    Type: Boolean
    Default value: true
  • start * Optional
    The value at the most left position
    Type: Number (Natural or decimal)
    Default value: 0
    Value examples: -1, -9.7, 14000, 200, 3.7
  • steps * Optional<
    Define the slider behaviour and selectionable values
    (No order required)
    Type: Array of Object{step,number}
    Default value: [{step:1,number:10}]
    Value examples: Go to --> Changing steps <--
  • values * Optional
    Slider pins initial values (No order required)
    but have to be one of the slider range values
    Type: Number or Array of Number()
    Default value: []
    Value examples: [6,0,3], 900, -6, -3.1

Methods

  • setValue(...)
    Dynamicaly set the slider values
    Parameter: Array or Number(Natural or decimal)
  • getValue()
    Return the slider values as an array
    Parameter: none
  • change()
    Reload - Simulate a value change
    Parameter: none

Some examples

Changing start and initial values

                        new LjSlider({
                            "elt": document.querySelector('.myclass'),
                            "start": -6,
                            "values": 0
                        });
                    
                        new LjSlider({
                            "elt": document.querySelector('.myclass'),
                            "start": -6
                        });
                    
<div class="myclass" data-val="0"></div>

Changing steps

                        new LjSlider({
                            "elt": document.querySelector('.myclass'),
                            "steps": [
                                {"step": 500, "number": 5}
                            ]
                        });
                    
                        new LjSlider({
                            "elt": document.querySelector('.myclass'),
                            "steps": [
                                {"step": 30, "number": 10},
                                {"step": 500, "number": 6},
                                {"step": 50, "number": 5}
                            ],
                            "values": 800
                        });

Natural step

                        new LjSlider({
                            "elt": document.querySelector('.myclass'),
                            "steps": [
                                {"step": 30, "number": 10},
                                {"step": 500, "number": 6},
                                {"step": 50, "number": 5}
                            ],
                            "sameStep": false,
                            "values": 800
                        });

More pins

                        new LjSlider({
                            "elt": document.querySelector('.myclass'),
                            "steps": [
                                {"step": 30, "number": 10},
                                {"step": 500, "number": 6},
                                {"step": 50, "number": 5}
                            ],
                            "pins": 4,
                            "values": [60, 210, 2800, 3400]
                        });

No green links

                        new LjSlider({
                            "elt": document.querySelector('.myclass'),
                            "steps": [
                                {"step": 30, "number": 10},
                                {"step": 500, "number": 6},
                                {"step": 50, "number": 5}
                            ],
                            "pins": 2,
                            "link": false,
                            "values": 1300
                        });

Changing handler

                        function showVal(vals){
                            console.log('Values are: ',vals)
                        }

                        new LjSlider({
                            "elt": document.querySelector('.myclass'),
                            "steps": [
                                {"step": 30, "number": 10}
                            ],
                            "pins": 2,
                            "handler": "showVal"
                        });
